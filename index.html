Description:

The Definitive Guide to HTML & CSS--Fully UpdatedWritten by a Web development expert, the fifth edition of this trusted resource has been thoroughly revised and reorganized to address HTML5, the revolutionary new Web standard. Annotated examples of correct markup and style show you how to use all of these technologies to build impressive Web pages.
